# 숙제 11.7 -- MPG 비디오 파일 서빙

## 문제

Tiny를 확장하여 MPG 비디오 파일을 서빙하도록 하라.

## 수정 내용

`get_filetype()` 함수에 `.mpg`와 `.mpeg` 확장자에 대한 MIME 타입 `video/mpeg`을 추가했다.

### 수정 전

```c
} else if (strstr(filename, ".jpg") || strstr(filename, ".jpeg")) {
    strcpy(filetype, "image/jpeg");
} else {
    strcpy(filetype, "text/plain");
}
```

### 수정 후

```c
} else if (strstr(filename, ".jpg") || strstr(filename, ".jpeg")) {
    strcpy(filetype, "image/jpeg");
} else if (strstr(filename, ".mpg") || strstr(filename, ".mpeg")) {
    strcpy(filetype, "video/mpeg");
} else {
    strcpy(filetype, "text/plain");
}
```

## 테스트 방법

```bash
# Tiny 서버 실행
$ ./tiny 8080

# 브라우저에서 접속
http://localhost:8080/sample.mpg
```

브라우저가 `Content-type: video/mpeg` 헤더를 받으면 비디오 플레이어로 재생을 시도한다.
