# 숙제 11.10 -- CGI adder용 HTML form 작성

## 문제

CGI adder 함수를 위한 HTML form을 작성하라.
form에는 사용자가 더할 두 숫자를 입력하는 텍스트 박스 두 개가 포함되어야 한다.
form은 GET 메서드를 사용하여 콘텐츠를 요청해야 한다.

## 구현

`adder-form.html` 파일을 Tiny 서버 디렉토리에 생성했다.

### 핵심 부분

```html
<form action="/cgi-bin/adder" method="GET">
    <label for="n1">Number 1:</label>
    <input type="text" id="n1" name="n1" value="15000">
    <br><br>
    <label for="n2">Number 2:</label>
    <input type="text" id="n2" name="n2" value="213">
    <br><br>
    <input type="submit" value="Add">
</form>
```

### 동작 원리

1. 사용자가 두 숫자를 입력하고 "Add" 버튼을 클릭한다.
2. 브라우저가 GET 요청을 생성한다:
   ```
   GET /cgi-bin/adder?n1=15000&n2=213 HTTP/1.1
   ```
3. Tiny의 doit()이 요청을 파싱한다.
4. parse_uri()가 "cgi-bin"을 감지하고 동적 콘텐츠로 판별한다:
   - filename = "./cgi-bin/adder"
   - cgiargs = "n1=15000&n2=213"
5. serve_dynamic()이 fork+execve로 adder를 실행한다.
6. adder가 QUERY_STRING에서 인자를 추출하고 결과를 출력한다.

## 테스트 방법

```bash
$ ./tiny 8080

# 브라우저에서
http://localhost:8080/adder-form.html
```

기본값(15000, 213)으로 제출하면 결과: 15000 + 213 = 15213
